# Support

Before opening an issue:

1. Run **Local N-Gram: Show Model Diagnostics** and copy the non-sensitive output.
2. Confirm the problem with the bundled starter model.
3. Include VS Code, extension, operating-system, and model-format versions.
4. Include a minimal code example with private identifiers removed.

Do not attach proprietary source code or private model files.

Open support requests in the
[public fork issue tracker](https://github.com/kevinski119/local-ngram-code-suggester/issues).
Upstream historical issues remain available at
[amest/vscode-ngram-code-suggester](https://github.com/amest/vscode-ngram-code-suggester/issues).
